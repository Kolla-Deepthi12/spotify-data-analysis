# Spotify Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on Spotify track and artist data to understand:
- Popularity patterns
- Artist influence
- Track characteristics
- Correlations between numeric features

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Univariate Analysis (Popularity, Duration)
- Bivariate Analysis (Popularity vs Followers)
- Group Analysis (Album Type, Artist)
- Time-based Trends
- Correlation Heatmap

##  Key Insights
- Artist popularity strongly correlates with follower count
- Track popularity has weak correlation with duration
- Album type influences popularity distribution
